# Raven Mood

An AI-powered code mood analyzer dashboard. Deploy-ready for Vercel.

## Deployment

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/raven-mood)

To deploy to Vercel:
1. Ensure your repository is pushed to GitHub.
2. Import the project in Vercel.
3. Add the `ANTHROPIC_API_KEY` environment variable.
4. Click deploy.

## Local Development
1. Install dependencies: `npm install`
2. Run development servers: `npm run dev`
3. Backend requires python dependencies in `apps/api/requirements.txt` to be installed locally.
